Note: If these directions confuse you, feel free to use the web version on the main webpage.
1) Requires Python 3.5.3 or greater.
2) If you have Windows, you can just double-click on the intention_repeater.py file to run in the File Explorer.
3) To run on Linux or MacOS, run in terminal in the folder of your python file: python3 intention_repeater.py
4) To automate, on Windows, with parameters (for v6.0+), example: intention_repeater.py YYYN8 "HH:MM:SS" "[Intentions or file with intents]"
5) To automate, on Linux, with parameters (for v6.0+), example: python3 intention_repeater.py YYYN8 "HH:MM:SS" "[Intentions or file with intents]"
6) First letter of YYYN8 could be Y or N. Y will utilize Gut/Heart/Mind coherence. It can be exhausting, but it puts gut/heart/mind in resonance and is stronger.
7) Second letter of YYYN8 could be Y or N. Y will clear energetic interference. It can be exhausting, but it clears energetic interference and is more efficient.
8) Third letter of YYYN8 could be Y or N. Y will use the Akashic Records. It can be very expansive and hard to focus down.
9) Fourth letter of YYYN8 could be Y or N. Y will use the most powerful Quasar. A strong power source, but can be draining. I used YYYN so you don't confuse it with the year.
10) The 5th number of YYYN8 is the intensity. On modern computers this can go up to 8. Celphones and less powerful PC's go up to 7.
11) The "HH:MM:SS" parameter determines how long to run for. If you want to manually stop it, put "00:00:00" or "Forever".
12) Finally, there is the intention/filename, separated by a space from the earlier parameters. If using a file, maintain the upper/lowercasing.
13) This is run in the terminal. For Linux, the filename is case sensitive. Though it doesn't access the file itself, and goes more by intention.
