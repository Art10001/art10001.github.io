The free, open-source Intention Repeater WAV Converter that made this encodes any file (Approx <10MB)
into a 96kHz Mono WAV File at 95% max volume.
It is not meant to sound good. But it allows you to feel whatever is going on in the picture.
The WAV Converter works best with images or text. Audio/Video will still work, but it won't sound pleasant.
Default Windows Usage: python intention_repeater_wav_converter.py
Default Linux Usage: python3 ./intention_repeater_wav_converter.py
Automated Windows Usage: python intention_repeater_wav_converter.py "HH:MM:SS" "Filename to Convert" "Output WAV Filename"
Automated Linux Usage: python3 ./intention_repeater_wav_converter.py "HH:MM:SS" "Filename to Convert" "Output WAV Filename"
The HH:MM:SS is the hours, minutes and seconds of the WAV file you would like to create.

Minimum Python 3.5.3 is required.

You can get the free, open-source here: Intention Repeater WAV Converter Python
